# MindMap Google-Ready Website

This is a proper static website version of your MindMap app. It includes:

- `index.html`
- `style.css`
- `script.js`
- `robots.txt` for Google crawling
- `sitemap.xml` for Google Search Console
- `favicon.svg`
- `netlify.toml` and `vercel.json` deployment config

## Run locally
Open `index.html` in your browser.

## Publish online
Drag this full folder into Netlify Drop, or upload it to Vercel/GitHub Pages.

## Important
After deployment, replace `https://YOUR-SITE-URL.netlify.app/` in `index.html`, `robots.txt`, and `sitemap.xml` with your real website URL.

Then submit the site to Google Search Console.
